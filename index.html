FEATURES

IPL KEY FEATURES

Multi-Feature

IPL can perform grayscale average, flip on x axis, image convolution, and so much more...

Multi-Threading

IPL can process hundreds of images concunrently. Thanks to its multi-threaded feature

Multi-Platfform

IPL library has a desktop version for Linux, Windows, and MAC workstations.

Multi-Optimized

IPL was built and optimized using web assembly. Code was later optimized through rigorous inspection

Tech Stack

IPL was built using Web Assembly. Desktop version was built using Electron.

IPL was originally written in C++ but then transpiled into JavaScript using emscripten for maximum optimization. The Desktop version is an Electron App which allows the flexibility of running it in the browser as well.

USING JAVASCRIPT LIBRARY IN CODE?

const Module = require('./ipl_min.js');

let base64s = (some comma-seperated image based64 string)

let images = new Module.ImageOperations();

images.createImages(base64s);

images.grayscale_avgs();

const response = images.encodeBytes(); // returns comma-seperated based64 string.
